Mbed Lab 2 - Control de LEDs con DigitalIO y BusIO

Práctica de sistemas embebidos con el framework Mbed sobre microcontrolador ARM.

Descripción

Control de 3 LEDs (rojo, amarillo, azul) mediante 4 botones, utilizando interrupciones con InterruptIn. Cada botón permite cambiar el estado de un LED o encenderlos todos simultáneamente.

Hardware

Microcontrolador ARM compatible con Mbed (NUCLEO / LPC1768)
4 botones en pines p5-p8
3 LEDs en pines p9-p11

Conceptos aplicados

InterruptIn
DigitalOut
GPIO
Manejo de interrupciones
Control de periféricos

Plataforma

Desarrollado y simulado en Mbed Simulator
